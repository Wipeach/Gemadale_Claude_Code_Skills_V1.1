# 房地产项目分析PPT生成器

这是一个用于自动化生成房地产项目分析PPT的技能。该技能集成了多个数据处理模块，能够从原始数据文件中提取信息，并自动生成专业的PowerPoint演示文稿。

## 功能特点

- **住房数据分析**：解析项目基本信息
- **土地数据分析**：解析土地相关信息  
- **供应数据分析**：分析房地产供应情况
- **成交数据分析**：分析成交数据并生成可视化图表
- **开盘数据分析**：分析开盘相关信息
- **户型图分析**：通过LLM分析户型图
- **客户分析**：分析客户数据并生成地域来源饼图
- **周边信息分析**：分析项目周边配套信息
- **PPT自动生成**：创建完整的5页专业PPT

## 使用方法

1. 准备项目数据文件，按照以下命名规范放在 `resources/working_data/{项目名称}_{日期}/` 目录下：
   - `{项目名称}_基本信息.txt`
   - `{项目名称}_土地信息.txt`
   - `{项目名称}_供应明细底表.xlsx`
   - `{项目名称}_成交分析结果.xlsx`
   - `{项目名称}_开盘信息.xlsx`

2. 运行主脚本：
   ```bash
   python scripts/main_pipeline.py
   ```

3. 输入项目名称，系统将自动处理所有数据并生成PPT文件。

## 依赖要求

- Python 3.7+
- python-pptx
- pandas
- openpyxl
- 其他相关数据处理库

## 输出文件

生成的PPT文件将保存在 `resources/working_data/{项目名称}_{日期}/` 目录下，文件名为 `{项目名称}_{日期}.pptx`。

## 技能集成

此技能可直接集成到Moltbot中，通过调用相应的工具函数来执行房地产项目分析和PPT生成任务。
